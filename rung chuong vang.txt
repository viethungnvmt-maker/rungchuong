# 🚀 YÊU CẦU TẠO ỨNG DỤNG WEB: RUNG CHUÔNG VÀNG ONLINE

## 📝 MÔ TẢ Ý TƯỞNG
Tạo ứng dụng web gameshow "Rung Chuông Vàng" trực tuyến cho giáo viên, hỗ trợ tối đa 100 học sinh tham gia đồng thời. Hệ thống tự động chấm điểm, xếp hạng thời gian thực và tạo không khí thi đấu sôi nổi như gameshow truyền hình.

---

## ⚡ CHỨC NĂNG CHÍNH

### 1. Quản lý Câu hỏi (Question Bank)
- ✅ CRUD câu hỏi: Trắc nghiệm 4 đáp án, Đúng/Sai, Điền từ
- ✅ Upload hình ảnh/video đính kèm câu hỏi
- ✅ Phân loại theo môn học, độ khó (Dễ/Trung bình/Khó)
- ✅ Import/Export câu hỏi từ file JSON hoặc Excel
- ✅ **AI Generator**: Dùng Gemini AI tạo tự động 10-20 câu hỏi từ chủ đề

### 2. Tạo Phòng Chơi (Game Room)
- ✅ Tạo phòng với mã PIN 6 số hoặc QR Code
- ✅ Chọn bộ câu hỏi (10-50 câu)
- ✅ Cài đặt: Thời gian/câu (10-60s), Điểm thưởng tốc độ, Chế độ loại trực tiếp
- ✅ Phòng chờ hiển thị danh sách học sinh đã vào (realtime)

### 3. Gameplay - Màn hình Giáo viên (Host)
- ✅ Hiển thị câu hỏi + 4 đáp án lớn trên màn hình chính
- ✅ Đồng hồ đếm ngược với hiệu ứng âm thanh
- ✅ Biểu đồ cột thời gian thực: Số người chọn từng đáp án
- ✅ Nút "Công bố đáp án" → Hiệu ứng pháo hoa + âm thanh chiến thắng
- ✅ Danh sách học sinh bị loại (nếu chế độ loại trực tiếp)
- ✅ **Tính năng "Cứu trợ"**: Chọn học sinh bị loại → Đưa câu hỏi phụ → Hồi sinh nếu đúng

### 4. Gameplay - Màn hình Học sinh (Player)
- ✅ Nhập tên + Avatar (chọn emoji hoặc upload ảnh)
- ✅ Nhập mã PIN → Vào phòng chờ
- ✅ Màn hình câu hỏi: 4 nút đáp án lớn, dễ bấm trên mobile
- ✅ Hiển thị thời gian còn lại + điểm hiện tại
- ✅ Feedback ngay lập tức: ✅ Đúng (màu xanh) / ❌ Sai (màu đỏ)
- ✅ Trạng thái: "Đang chơi" / "Bị loại" / "Được cứu trợ"

### 5. Bảng Xếp Hạng (Leaderboard)
- ✅ Top 10 học sinh điểm cao nhất
- ✅ Công thức điểm: `Điểm = (Đúng/Sai) × (1000 - Thời gian trả lời × 10)`
- ✅ Hiển thị: Vị trí, Tên, Avatar, Điểm, Số câu đúng
- ✅ Animation khi thay đổi vị trí
- ✅ Podium 3 người chiến thắng với hiệu ứng vương miện

### 6. Báo Cáo & Thống Kê
- ✅ Tổng quan: Tổng số người chơi, Tỷ lệ đúng trung bình, Thời gian chơi
- ✅ Chi tiết từng học sinh: Điểm, Câu đúng/sai, Thời gian trung bình
- ✅ Phân tích câu hỏi: Câu nào khó nhất (tỷ lệ sai cao)
- ✅ Export PDF/Excel báo cáo

### 7. Âm thanh & Hiệu ứng
- ✅ Nhạc nền gameshow (có thể tắt/bật)
- ✅ Âm thanh đếm ngược (tick-tock 5 giây cuối)
- ✅ Tiếng chuông "Ting!" khi trả lời đúng
- ✅ Tiếng "Buzz" khi trả lời sai
- ✅ Pháo hoa + confetti khi công bố người chiến thắng

// Tính năng: Tạo tự động câu hỏi từ chủ đề
async function generateQuestionsWithAI(topic, count, difficulty) {
  const prompt = `
Tạo ${count} câu hỏi trắc nghiệm về chủ đề "${topic}" với độ khó ${difficulty}.

Yêu cầu:
- Mỗi câu có 4 đáp án
- Chỉ 1 đáp án đúng
- Phù hợp với học sinh THCS/THPT
- Có giải thích ngắn gọn

Trả về JSON format:
[
  {
    "content": "Câu hỏi...",
    "options": ["A", "B", "C", "D"],
    "correctAnswer": 0,
    "explanation": "Giải thích..."
  }
]
`;

  const response = await callGeminiAI(prompt);
  return JSON.parse(response);
}

// Gemini API Call với fallback
const GEMINI_MODELS = ['gemini-2.0-flash-exp', 'gemini-1.5-flash', 'gemini-1.5-pro'];

async function callGeminiAI(prompt, modelIndex = 0) {
  const API_KEY = localStorage.getItem('gemini_api_key');
  if (!API_KEY) {
    showToast('⚠️ Chưa cài đặt API Key!', 'warning');
    return null;
  }
  
  try {
    const response = await fetch(
      `https://generativelanguage.googleapis.com/v1beta/models/${GEMINI_MODELS[modelIndex]}:generateContent?key=${API_KEY}`,
      {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({
          contents: [{ parts: [{ text: prompt }] }],
          generationConfig: {
            temperature: 0.7,
            maxOutputTokens: 8192
          }
        })
      }
    );
    
    if (response.status === 429 && modelIndex < GEMINI_MODELS.length - 1) {
      console.log(`Model ${GEMINI_MODELS[modelIndex]} bị giới hạn, chuyển sang ${GEMINI_MODELS[modelIndex + 1]}`);
      return callGeminiAI(prompt, modelIndex + 1);
    }
    
    if (!response.ok) throw new Error(`HTTP ${response.status}`);
    
    const data = await response.json();
    return data.candidates?.[0]?.content?.parts?.[0]?.text || '';
    
  } catch (error) {
    if (modelIndex < GEMINI_MODELS.length - 1) {
      return callGeminiAI(prompt, modelIndex + 1);
    }
    showToast('❌ Lỗi API: ' + error.message, 'error');
    return null;
  }
}
